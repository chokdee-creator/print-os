ตั้งค่า Google Sheet

นำโค้ด Apps Script จากไฟล์ที่แนบไปวางใน Google Apps Script แล้ว Deploy เป็น Web app จากนั้นนำ URL มาวางที่นี่ รายการใหม่จะถูกส่งเข้า Sheet อัตโนมัติ

1. เปิด Google Sheet ใหม่
2. Extensions > Apps Script
3. วางโค้ดจากไฟล์ google-apps-script.js
4. Deploy > New deployment > Web app
5. Execute as: Me, Who has access: Anyone
6. Copy Web app URL มาใส่ในช่องนี้
